## ⚡ THE FIVE CRITICAL RULES

Memorize these immediately:

### Rule 1: Events from Shared Contracts ONLY
All events must originate from `shared/contracts/events`
No service may define local events

### Rule 2: SDKs from Packages ONLY
Use `packages/kafka-sdk`, `packages/event-bus`, etc.
Never import raw libraries directly

### Rule 3: Platform Abstractions Required
Use `platform/event-bus`, `platform/saga`, etc.
Never implement custom frameworks

### Rule 4: Reference Architecture Pattern
Follow `services/auth-service/` structure exactly
Same in every service: domain, application, infrastructure, transport

### Rule 5: No Cross-Service Database Writes
Never update another service's tables
Always use gRPC or events
